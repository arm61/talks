Using programming as a chemistry teaching tool
an_drewmcc

who?

  • First year natural science students
  • Mixture of chem/phys and bio/chem


how?

  • Jupyter notebooks provides a user-friendly environment
  • Students can access the notebooks online via MyBinder
an_drewmcc

why?

  • Programming requires the mathematics to be broken down
  • Can provide instantenous feedback in problem based learning

tools we've developed

  • Bath has dedicated computational chemistry labs
  • Needed a tool to introduce molecular dynamics/Monte-Carlo
an_drewmcc

pylj

  • Open-source, Python library to engage students in classical atomistic simulation
  • Can be used to introduce molecular dynamics or Monte-Carlo simulation
  • Or used to show how simulation can describe physical systems
an_drewmcc

pylj

  • Paper in Journal of Open-Source Education, 1(2), 19. DOI: 10.21105/jose.00019
  • MIT Licence; source code available on GitHub
  • Bath physics PDRA forked it to make pysing
